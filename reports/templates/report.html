Platform Architecture · AgentPulse
Methodology & Architecture
Eight open-source tools wired together — from user query to scored evaluation result. Every component runs locally with no cloud API keys required.
Scoring Method A
Formula-Based Scoring
Used for RAG metrics. Mathematical algorithms verify the response against retrieved context. Scores are computed deterministically — no LLM judgment involved.

Example: Faithfulness = supported claims ÷ total claims
Scoring Method B
GEval — LLM as Judge
Used for domain and safety metrics. A second LLM reads the response and scores against plain-English criteria. Captures tone, actionability, task resolution.

Judge: Ollama llama3.2:3b — no cloud key needed.
